# BLE-система с несколькими устройствами (Raspberry Pi Pico W)

📌 Описание проекта

В данном проекте реализована система беспроводного обмена данными по протоколу Bluetooth Low Energy (BLE) между несколькими микроконтроллерами Raspberry Pi Pico W.

Система состоит из:

* одного центрального устройства (Master — PICO_A)
* двух периферийных устройств (Slave — PICO_B и PICO_C)

Центральное устройство поочерёдно подключается к каждому периферийному, считывает данные и отображает их на экране.

        ┌──────────────┐
        │   PICO_B     │
        │  (Slave)     │
        │  B:значение  │
        └──────┬───────┘
               │ BLE
               │
        ┌──────▼───────┐
        │   PICO_A     │
        │  (Master)    │
        │  Экран       │
        └──────▲───────┘
               │
               │ BLE
        ┌──────┴───────┐
        │   PICO_C     │
        │  (Slave)     │
        │  C:значение  │
        └──────────────┘

Принцип работы

Работа системы происходит циклически:

1. PICO_A начинает сканирование BLE-устройств
2. Находит устройство (например, PICO_B)
3. Подключается к нему
4. Находит сервис и характеристику
5. Считывает данные (например, B:230)
6. Отключается
7. Повторяет те же действия для PICO_C
8. Процесс повторяется по кругу
